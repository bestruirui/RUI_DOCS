## 查看CPU频率
```bash
watch grep \'cpu MHz\' /proc/cpuinfo
```
## 查看CPU温度
```bash
cat /sys/class/hwmon/hwmon*/temp*input
```
## 解压缩
```bash
 #解压
tar -zxvf test.tar.gz        
#压缩
tar -zcvf test.tar.gz ./test/ 
```
## 清理日志
```shell
#只保留两天的，其余全部删除
journalctl --vacuum-time=2d  

#删除bash历史
rm -rf ~/.bash_history
history -c
```
