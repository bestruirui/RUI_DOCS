1. 先准备一个学信网的证明[学信档案 ](https://my.chsi.com.cn/archive/bab/index.action)
2. 在查看页面使用浏览器直接翻译成英语
3. 翻译完成后按`F12`在元素页面把`full-time`替换成`remote study`
